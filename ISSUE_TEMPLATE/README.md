# Issue Templates

This folder defines organization-wide GitHub issue templates for ActiumDigital repositories.

## Files

- `bug_report.md` — Bug reporting template.
- `feature_request.md` — Feature proposal template.
- `documentation.md` — Documentation issue template.
- `config.yml` — Issue form behavior (`blank_issues_enabled`, contact links).

## Included Conventions

- Every template includes a **Redmine Reference** field.
- Standard labels are prefilled by template type.
- Structured sections ensure reproducible and actionable issues.

## Notes

Repository-level issue templates can override these organization defaults.
